Фотогалереи из социальной сети 'вконтакте' vk.com

== Description ==

Открывает фотогалереи сети "вконтакте", реализована функций кеширования и удобного, адаптивного просмотра фотографий.
Поддержка языков: ENG,RUS

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
